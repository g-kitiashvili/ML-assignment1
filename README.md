##  Feature Selection

### 1. рЃњрЃљрЃбрЃћрЃАрЃбрЃћрЃЉрЃБрЃџрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў:

#### **Univariate Selection (SelectKBest)**
```python
selector = SelectKBest(score_func=f_regression, k=50)
```
- **рЃърЃарЃўрЃюрЃфрЃўрЃърЃў**: рЃДрЃЋрЃћрЃџрЃљ рЃюрЃўрЃерЃюрЃўрЃА рЃўрЃюрЃЊрЃўрЃЋрЃўрЃЊрЃБрЃљрЃџрЃБрЃарЃў рЃЎрЃЮрЃарЃћрЃџрЃљрЃфрЃўрЃљ target-рЃЌрЃљрЃю
- **рЃерЃћрЃЊрЃћрЃњрЃў**: рЃерЃћрЃарЃЕрЃћрЃБрЃџрЃў 50 рЃБрЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃћрЃАрЃў рЃюрЃўрЃерЃљрЃюрЃў

#### **Recursive Feature Elimination (RFE)**
```python
estimator = RandomForestRegressor(n_estimators=50, random_state=42)
selector = RFE(estimator, n_features_to_select=50)
```
- **рЃърЃарЃўрЃюрЃфрЃўрЃърЃў**: рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА importance scores-рЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў рЃћрЃџрЃўрЃЏрЃўрЃюрЃљрЃфрЃўрЃљ
- **рЃерЃћрЃЊрЃћрЃњрЃў**: Random Forest-рЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ 50 рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃюрЃўрЃерЃљрЃюрЃў

#### **L1-based Selection (Lasso)**
```python
lasso = Lasso(alpha=0.01, random_state=42)
selector = SelectFromModel(lasso)
```
